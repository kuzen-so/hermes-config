# 形象优化系统模板

**来源：** 2026-05-05 对话，用户要求将饮食/训练/形象维护协议写入 SOUL.md
**适用：** 当用户提及减脂增肌、形象管理、面部优化、视频出镜准备时参考

---

## 目标

体脂率优先于体重数字，面部轮廓清晰，视觉资产固定可识别。

## 饮食协议

- 定时定量，到点执行，不判断饿不饿
- 蛋白质优先，蛋白粉+鸡蛋为主，极简烹饪
- 两餐或三餐，睡前3小时不进食

## 训练协议

- 每周3次力量训练，每次45分钟，到点走
- 上肢优先（胸肩背），暂不加腿
- 不做有氧，减脂靠饮食缺口

## 形象维护

- 胡子每2-3天刮净，不留中间态
- 黑眼圈靠睡眠+热敷，零产品
- 发型定期补色修剪，酒红不褪成橘黄
- 穿搭固定3件上衣轮换，黑/白/藏蓝，零决策

## 执行规则

蛋白粉到货后启动，不提前囤鸡蛋，不追求完美数字，先跑起来。

- 荞麦面：每周300g
- 蔬菜：3种，一次焯水3天量

## 关键决策记录

1. **不买增肌粉(Gainer)**：碳水太高，反向减脂。必须买乳清蛋白(Whey Protein)。
2. **分离乳清 vs 浓缩乳清**：分离乳清蛋白质更高(90%)但原味难喝；浓缩乳清(82%)有口味，可执行。最终选浓缩乳清巧克力味。
3. **蛋黄控制**：每天2个全蛋+6个蛋白，胆固醇可控。
4. **不做有氧**：减脂靠饮食缺口，有氧浪费时间。
5. **暂不加腿**：上肢优先，下肢"看见效果再练"（用户决策，已记录）。
