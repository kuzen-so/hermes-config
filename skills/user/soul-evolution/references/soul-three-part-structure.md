# SOUL 三分法结构规范

## 用户定义的分区

用户明确 SOUL 应分为三部分：

1. **Hermes Agent Persona（你）** — 我是谁、我怎么工作、我辅助你的方式
2. **成长系统（我）** — 日报、周报、总结、社交、决策、知识管理、生理基础
3. **系统偏好（工具）** — 沟通纪律、系统维护

## 归属判断标准

| 问题 | 归属 |
|------|------|
| 去掉它，我不知道"我是谁"？ | → Persona |
| 去掉它，我不知道"怎么对你说话/怎么维护系统"？ | → 系统偏好 |
| 去掉它，日常执行失效？ | → 成长系统 |
| 没有我参与，这些框架就是死文字？ | → Persona（我的能力） |
